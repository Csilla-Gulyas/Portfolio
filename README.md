# Portfolio Website

Ez a repository a személyes portfólióoldalamat tartalmazza, amely bemutatja a projektjeimet, technológiai tudásomat és fejlesztői szemléletemet.

Live demo: https://csilla-gulyas.github.io/Portfolio/

---

## About the Project

A portfólió célja, hogy átlátható képet adjon a webfejlesztés területén szerzett tapasztalataimról, valamint az eddigi egyéni és tanulmányi projektjeimről.  
Az oldal elsősorban junior / gyakornoki pozíciókra való jelentkezéshez készült.

---

## Features

- Bemutatkozó szekció
- Projektek részletes leírással és linkekkel
- Használt technológiák áttekintése
- Kapcsolati és elérhetőségi információk
- Reszponzív megjelenés

---

## Technológiák

- HTML5
- CSS3
- JavaScript

## Setup / Usage

A repository klónozása után a fájlok közvetlenül megnyithatók böngészőben.
